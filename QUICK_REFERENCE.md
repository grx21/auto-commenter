## Troubleshooting Quick Fixes

| Problem | Quick Fix |
|---------|-----------|
| "Can't find file" | Check file path and name |
| "Page won't load" | Restart Claude, check internet |
| "Not logged in" | Open Reddit in browser, log in |
| "Rate limited" | Wait 30 min, reduce quota |
| "Doesn't sound like me" | Update personalization file |

## Safety Limits

### Recommended Daily Quotas
- **Starting out**: 3-5 comments/day
- **Established**: 10-15 comments/day
- **Maximum**: 21 comments/day (7 subreddits × 3)

### Red Flags to Watch
- Comments getting downvoted
- Community members questioning authenticity
- Rate limit warnings
- Account restrictions

**If you see these, STOP and review your approach.**

## Emergency Actions

### If Your Account Gets Flagged

1. **STOP all automation immediately**
2. Review recent comments for issues
3. Check Reddit DMs for warnings
4. Reduce or pause automation for 1-2 weeks
5. Focus on manual, high-quality engagement
6. Gradually resume with lower quotas

### If Comments Get Downvoted

1. Review the comment and post
2. Check if you misunderstood the post
3. Update personalization if style is off
4. Reduce activity temporarily
5. Focus on fewer, higher-quality comments
